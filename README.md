# Experimenting with Sockets

## Echo Server & Client
TCP Echo Server and Client, written in C. Server can handle multiple client connections concurrently by using epoll(). 

### Build
- run ```make```

### Usage
- run ```./server```
- run (up to 128) ```./client``` instances
  
Enjoy the echo
